Cesar vs Vigenere vs RSA

Explore trois methodes de chiffrement : du simple decalage a la cryptographie asymetrique moderne.

PARAMETRES DE CLE
DECALAGE (CLE)
3
RESULTAT
i
Logique Mathematique

Le chiffre de Cesar decale chaque lettre d'un nombre fixe.

FORMULE DE CHIFFREMENT
L' = (L + 3) mod 26
L = Lettre Originale
mod = Reste division
i
Logique Mathematique

Le chiffre de Vigenere utilise une cle pour varier le decalage de chaque lettre.

FORMULE DE CHIFFREMENT
C[i] = (M[i] + K[i]) mod 26
M = Message
K = Cle (repetee)
i
Cryptographie Asymetrique

RSA utilise deux cles : une publique pour chiffrer et une privee pour dechiffrer.

FORMULES
C = M^e mod n
M = C^d mod n
n (p x q) 3233
e (public) 17
d (prive) 2753
